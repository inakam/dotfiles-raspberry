## 8. 文末表現

### 単調さ解消のバリエーション

| パターン | 例                                                    |
| :------- | :---------------------------------------------------- |
| 体言止め | 「〜という特徴があります。」→「〜という特徴。」       |
| 倒置法   | 「私はそれが好きです。」→「好きなんだよね、それが。」 |
| 疑問形   | 「重要です。」→「大事じゃないですか？」               |
| 断定     | 「〜と思われます。」→「〜だ。」                       |
| 口語調   | 「〜ですね。」→「〜だよね。」                         |

### 「です・ます」連続の回避

```
# Before
AIは便利です。多くの人が使っています。効率が上がります。

# After（バリエーション）
AIは便利だ。多くの人が使っている。効率？ 確実に上がる。
```

## 9. 漢語から和語への変換

音読み熟語（漢語）を訓読み（和語）に開くことで親近感を高める。

| 漢語   | 和語               |
| :----- | :----------------- |
| 効率化 | 無駄をなくす       |
| 最適化 | ぴったりにする     |
| 構築   | 作る               |
| 提供   | 渡す               |
| 確認   | 見る、チェックする |
| 実施   | やる               |
| 達成   | できる             |
| 実現   | できるようにする   |
| 活用   | 使う               |
| 促進   | 進める             |
