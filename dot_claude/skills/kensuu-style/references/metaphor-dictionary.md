# けんすう流メタファー辞典

抽象概念を「物理的なモノ」に変換するための辞書。

## メタファー変換事例集

| 抽象的な概念             | けんすう流メタファー         | 効果・解説                                                                |
| :----------------------- | :--------------------------- | :------------------------------------------------------------------------ |
| 行動経済学・習慣形成     | **摩擦（マサツ）**           | スマホ中毒対策→アプリ起動の手間を「摩擦」と表現。物理学のイメージで直感化 |
| 自己効力感・レジリエンス | **ドラクエ（RPG）**          | 人生をゲームに見立て、「失敗」を「経験値稼ぎ」「イベント」と再定義        |
| リソース配分・合理性     | **紙コップの使い捨て**       | 食器洗いのコストvs紙コップ代。極端な具体例で合理主義を印象付ける          |
| キャリア形成・適性       | **攻略本・レシピ**           | 成功者のやり方をトレース。「ハンバーグに独創性は要らない」                |
| 人間関係の分布           | **2:2:6の法則**              | 10人中2人味方、2人敵、6人どちらでもない。数値化で客観視                   |
| 自己分析                 | **棚卸し・在庫**             | 「棚卸ししても棚に在庫がない」→自己分析の限界を言語化                     |
| 目標設定                 | **チェックポイント**         | ゲームのセーブポイントのように小さな達成感を設計                          |
| 認知負荷                 | **脳内メモリ**               | ワーキングメモリを「脳内メモリ」と表現し、容量制限を直感化                |
| 情報過多                 | **文字の壁（Wall of Text）** | 長文段落を「壁」と表現。視覚的圧迫感をイメージ化                          |

## メタファー変換の原則

### 1. 身体性を持たせる

```
# Before
「心理的障壁が高い」

# After
「ハードルが高い」「摩擦がある」「重い」
```

### 2. 日常のモノに置き換える

```
# Before
「効率的な情報処理」

# After
「コンビニ弁当のように手軽」
「自動販売機のようにすぐ出てくる」
```

### 3. ゲーム用語を活用

| 抽象概念   | ゲーム用語               |
| :--------- | :----------------------- |
| 失敗       | ゲームオーバー、リトライ |
| 経験       | 経験値、レベルアップ     |
| 困難       | ボス戦、イベント         |
| スキル習得 | スキルツリー、装備       |
| 目標       | クエスト、ミッション     |
| 成長       | ステータス上昇           |
| 選択       | ルート分岐               |
| 再挑戦     | コンティニュー           |

### 4. 物理法則を比喩に使う

| 物理概念 | 比喩的使用                 |
| :------- | :------------------------- |
| 摩擦     | 行動の障壁、抵抗           |
| 慣性     | 習慣の継続力               |
| 重力     | 引き戻される力、デフォルト |
| 加速度   | 成長のスピード変化         |
| 圧縮     | 情報の凝縮                 |

## 活用例

### キャリア論

```
# Before
「自分のやりたいことがわからない」

# After
「棚卸ししようにも、棚に在庫がないんですよね」
```

### 習慣形成

```
# Before
「習慣化するには環境を整えることが重要です」

# After
「摩擦を減らすんです。ジム行くのに着替えが必要だと、それだけで面倒になる」
```

### 失敗の受容

```
# Before
「失敗を恐れずに挑戦することが大切です」

# After
「ドラクエで全滅しても、セーブポイントからやり直せるじゃないですか。人生も同じです」
```

### 効率化

```
# Before
「時間の効率化を図るべきです」

# After
「紙コップを使うんです。洗う時間を0にする。そのほうが精神的にも楽」
```

## メタファー生成のコツ

1. **五感で感じられるモノ**を選ぶ
2. **誰もが知っている**モノを使う
3. **極端な具体例**で印象を強める
4. **ネガティブな側面**も正直に含める
5. **自分の体験**と紐づける
